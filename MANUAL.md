# 점검이력 변환기 사용 설명서

처음 쓰는 분도 그대로 따라 하면 됩니다. 천천히 한 줄씩 읽어보세요.

---

## 0. 이 앱은 뭐 하는 거예요?

복사해온 **점검/접수 내용**을 깔끔한 보고 양식으로 **자동으로 바꿔주는** 앱이에요.
바뀐 결과를 복사해서 메신저(카톡 등)에 붙여넣으면 끝입니다.

핵심 흐름은 딱 4단계예요.

> **① 위 칸에 원본 붙여넣기 → ② 자동으로 결과가 만들어짐 → ③ 빈 칸 채우기 → ④ 복사하기**

---

## 1. 먼저 맨 위에서 "탭" 고르기

화면 맨 위에 단추 4개가 있어요. 지금 하려는 일에 맞는 걸 누르세요.

- **점검** : 복합기/프린터 점검 내용을 정리할 때 (한 곳에 여러 대 있어도 됨)
- **미양식** : 양식이 안 갖춰진 접수 글을 정리할 때 (보통 1대)
- **청정기** : 공기청정기 점검 정리할 때
- **삼성노트** : 스케줄 제목만 번호대로 정리할 때

> 색깔이 들어온 단추가 지금 켜져 있는 탭이에요.

---

## 2. "원본 붙여넣기" 로 원본 넣기

화면의 **📋 원본 붙여넣기 / 입력** 단추를 누르면 **팝업 창**이 떠요.

- 팝업 안에서 복사한 내용을 붙여넣거나, **📋 붙여넣기** 단추를 누르세요.
- 다 넣었으면 **확인 (변환)** 을 누릅니다.

그러면 팝업이 닫히고 **결과가 저절로 만들어져요.** (아래쪽에 결과 칸이 생깁니다.)

> 원본 창은 평소엔 화면에 안 보여서 스크롤이 편해요. 원본은 저장되지 않아 다음에 열면 비어 있어요(결과·작성한 내용은 그대로 남아요).

---

## 3. "처리내용 입력" 칸 채우기

원본 아래에 빈 칸들이 쭉 나와요. 점검하면서 확인한 내용을 채우면 됩니다.
**칸을 채우면 아래 결과가 바로바로 바뀝니다.**

채우는 항목들:

- **작성자** : 칸을 누르면 팀(팀장/A/B/C/D)이 나와요. 내 이름을 고르세요.
  (한 번 고르면 다음에도 기억해요.)
- **레벨** : (미양식 탭에서만 보임)
- **처리내용** : 무슨 일을 했는지 적기. 예) `정기점검`, `헤드 청소`
- **매수** : 흑 / 컬 / 큰컬 / 합 칸에 숫자 적기
- **잔량 (%)** : K · C · M · Y · 폐통 칸에 숫자 적기
- **여분** : 남은 토너를 적는 칸. 원래 내용이 그대로 들어와 있으니 숫자만 고치면 돼요.
- **한틴이카유무 / 주차비지원유무** : 단추(칩)를 눌러 빠르게 고르거나, 칸에 직접 적기
- **특이사항** : 특별한 일이 있었으면 적기
- **부품신청 / 자가신청** : 평소엔 접혀 있어요. 필요할 때 **펼치기 ▼** 를 눌러 적기
- **도착 시간 / 소요 시간** : 시/분을 고르고, 소요 시간은 `+1 +5 +10 +30 +60` 단추를 눌러 더하기

> 💡 칸이 비어 있으면 원래 값이 그대로 남고, **내가 적은 것만 결과에 바뀌어 들어가요.**

---

## 4. 기기가 여러 대일 때 (점검 탭)

한 곳에 복합기가 2대 이상이면, 폼 위쪽에 **"기기 선택"** 칸이 생겨요.

- `1. 3층 왼쪽 / 모델명 / ...` 처럼 기기마다 이름이 나와요.
- **2번 기기**를 고르면, 화면 아래 결과도 **2번 기기 위치로 따라가요.**
- 기기를 바꿔가며 각각 채우면 됩니다. (먼저 채운 1번은 그대로 저장돼요.)

---

## 5. 아래 "결과" 확인하고 고치기

화면 **맨 아래에 결과 칸**이 항상 붙어 있어요.

- 위아래로 **스크롤**해서 전체 내용을 볼 수 있어요.
- 결과 칸 안의 글자도 **직접 고칠 수 있어요.** (등급·업체명 같은 걸 바로 수정 가능)
- 지금 고르고 있는 기기는 색이 연하게 칠해져서 구분돼요.

---

## 6. 복사하기 (마지막)

다 됐으면 맨 아래 단추 줄에서

- **📋 복사** 를 누르세요. → 전체 결과가 복사됩니다.
- 메신저에 가서 **붙여넣기** 하면 끝!

---

## 7. 다시 시작하고 싶을 때

- 맨 아래 **초기화** 단추를 누르면 모든 칸이 비워져요.

---

## 8. 안심하세요 (자동 저장)

- 적던 내용은 **자동으로 저장돼요.**
- 실수로 앱을 닫거나, 다른 화면 갔다 와도 **그대로 남아 있어요.**
- 완전히 지우고 싶을 때만 **초기화**를 누르면 됩니다.

---

## 9. (관리자용) 작성자 이름 추가·삭제

작성자 이름 목록은 `src/authors.ts` 파일에 들어 있어요.
이름을 바꾸려면:

1. GitHub에서 `src/authors.ts` 파일을 연다.
2. 해당 팀 줄에 이름을 추가하거나 지운다. 예) `A: ["김정민", "심태현", "새이름"]`
3. 저장(커밋)하면 1~2분 뒤 모든 사람 화면에 반영된다.

---

### 한눈에 보는 순서

1. 위에서 **탭** 고르기
2. **원본 입력**에 붙여넣기 (자동 변환됨)
3. **작성자** 고르기
4. (여러 대면) **기기 선택** 후 칸 채우기
5. 아래 **결과** 확인 / 필요하면 고치기
6. **📋 복사** → 메신저에 붙여넣기
