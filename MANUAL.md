# 📄 점검이력 변환기 사용 설명서

**처음이어도 그대로 따라 하면 됩니다.**
복사한 점검 내용을 깔끔한 보고 양식으로 자동으로 바꿔주는 앱이에요.

</div>

---

## ⭐ 한눈에 보는 순서 (이것만 기억!)

| 순서 | 할 일 |
|:---:|:---|
| **1** | 맨 위에서 **탭** 고르기 |
| **2** | **원본 붙여넣기** → 팝업에 붙여넣고 **확인** |
| **3** | **작성자** 고르고 빈 칸 채우기 |
| **4** | 맨 아래 **결과** 확인 (필요하면 직접 고치기) |
| **5** | **📋 복사** → 메신저에 붙여넣기 |

> 💡 변환 단추는 따로 없어요. 붙여넣고 확인하면 **결과가 저절로** 만들어집니다.

---

## 1️⃣ 탭 고르기

화면 맨 위 단추 4개 중 지금 할 일에 맞는 걸 누르세요.

| 탭 | 언제 쓰나요 |
|:---|:---|
| **점검** | 복합기·프린터 점검 (한 곳에 여러 대 OK) |
| **미양식** | 양식이 안 갖춰진 접수 글 정리 (보통 1대) |
| **청정기** | 공기청정기 점검 |
| **삼성노트** | 스케줄 제목만 번호대로 정리 |

> 색이 칠해진 단추가 지금 켜진 탭이에요.

---

## 2️⃣ 원본 붙여넣기

1. **📋 원본 붙여넣기 / 입력** 단추를 누르면 **팝업**이 떠요.
2. 팝업 안에 복사한 내용을 붙여넣어요. (**📋 붙여넣기** 단추를 눌러도 됨)
3. **확인 (변환)** 을 누르면 팝업이 닫히고 **결과가 만들어져요.**

> - 잘못 넣었으면 팝업의 **초기화** 단추로 지우고 다시 붙여넣으면 돼요.
> - 원본은 저장되지 않아 **다음에 열면 비어 있어요.** (작성한 결과·내용은 그대로 남아요.)

---

## 3️⃣ 칸 채우기

점검하면서 확인한 내용을 채우면 됩니다. **채우면 아래 결과가 바로바로 바뀌어요.**

- **작성자** — 칸을 누르면 팀(팀장/A/B/C/D)이 나와요. 내 이름 선택 *(다음에도 기억함)*
- **처리내용** — 무슨 일을 했는지. 예) `정기점검`, `헤드 청소`
- **매수 / 잔량(%)** — 숫자 직접 입력
- **여분** — 원래 내용이 들어와 있어요. **숫자만 고치면** 됩니다.
- **한틴이카 / 주차비** — 단추로 빠르게 고르거나 칸에 직접 입력
- **부품신청 / 자가신청** — 평소엔 접혀 있어요. 필요할 때 **펼치기 ▼**
- **도착 / 소요 시간** — 시·분 고르기, 소요시간은 `+1 +5 +10 +30 +60` 단추로 더하기

> 💡 칸이 비어 있으면 원래 값이 그대로 남고, **내가 적은 것만** 결과에 반영돼요.

---

## 4️⃣ 기기가 여러 대일 때 (점검 탭)

한 곳에 복합기가 2대 이상이면 폼 위쪽에 **"기기 선택"** 칸이 생겨요.

- `1. 3층 왼쪽 / 모델명 / …` 처럼 기기마다 이름이 나와요.
- **2번 기기**를 고르면 아래 결과도 **2번 기기로 따라가요.**
- 기기를 바꿔가며 각각 채우면 됩니다. *(먼저 채운 건 그대로 저장돼요.)*

---

## 5️⃣ 결과 확인하고 복사하기

- 화면 **맨 아래 결과 칸**은 위아래로 넘겨 볼 수 있어요.
- 결과 글자를 **직접 고칠 수도** 있어요. *(등급·업체명 같은 것도 바로 수정)*
- 다 됐으면 맨 아래 **📋 복사** → 메신저에 **붙여넣기**. 끝!
